# **分支(branch)和标签(tag)的区别**

---

|      特性      |                       分支(branch)                        |               标签(tag)                |
| :------------: | :-------------------------------------------------------: | :------------------------------------: |
|    **本质**    |           指向提交记录的**可移动指针**（动态）            | 指向特定提交记录的**固定指针**（静态） |
|    **用途**    |          并行开发（如功能开发、bug 修复）、迭代           |  标记重要版本（如 v1.0、v2.1 发布版）  |
| **是否可更新** |             可以继续提交代码，指针会向前移动              |   一旦创建，指针固定，不会随提交移动   |
|  **生命周期**  | 临时 / 长期（如 main 分支长期存在，feature 分支用完可删） |  永久保留（标记重要版本，一般不删除）  |
|  **常用示例**  |         main、develop、feature/login、hotfix/001          |        v1.0.0、v2.1.1、beta-0.9        |
